### 介绍

-   一个基于 `AutoHotKey` 编写的一个小工具
-   实现实时于光标处显示当前输入法中英文状态 (屏幕边缘隐藏)

### 使用示例

![demo](/demo.gif)

### 使用
-   下载并运行 `InputTip.exe` 即可

### 兼容情况

-   可用
    -   微软/搜狗/百度
-   不可用
    -   讯飞
### 注意

-   当没有任何可输入的焦点时(如桌面)，显示框会隐藏
